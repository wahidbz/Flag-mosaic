# Flag Mosaic – Assets

## Audio files (WAV data encoded as .mp3 stubs)
- bg-zen.mp3   – Background zen music loop (5s, low freq)
- click.mp3    – Tile click sound (0.1s, 800Hz)
- match.mp3    – Match chime (0.5s, 659Hz)  
- error.mp3    – Error buzz (0.3s, 150Hz)
- win.mp3      – Win fanfare (1.2s, 784Hz)

Replace these files with real MP3s for production.

## Sparkle assets
- sparkle.css  – CSS animation keyframes
- sparkle.svg  – Star icon for particles

## Notes
- All flag emojis are rendered via OS emoji fonts (no image files needed)
- 194 sovereign countries included (Israel excluded per spec)
- Particle effects use HTML5 Canvas (particleCanvas element)
